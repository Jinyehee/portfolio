GPS 트래킹 기반 아트 러닝 어플

사용자의 GPS 경로를 실시간으로 추적하고 이를 예술적인 러닝 아트로 표현하는 팀 프로젝트입니다. 프론트엔드는 React, 백엔드는 NestJS로 구축하였으며, 지도 시각화에는 Kakao Map API를 활용했습니다.

  • 기간: 2025.07.01 ~ 2025.07.24
  • 역할: 백엔드 + GPS 경로 추척 기능
  • 사용 언어: REACT, TYPESCRIPT, NESTJS, MYSQL, AWS
  • 기능: 회원가입, 사용자 GPS 추척, 러닝 아트 저장, 러닝 아트 따라그리기, 커뮤니티
    • - 실시간 GPS 추적 및 경로 기록

      사용자의 현재 GPS 위치를 지속적으로 추적하여, 이동 경로를 지도 상에 실시간으로 시각화합니다.
      데이터를 바탕으로 러닝 아트를 그릴 수 있고 다른 사람의 러닝 경로를 따라가는 기능도 이용할 수 있습니다.

    • - 러닝 아트 시각화 및 커뮤니티 공유

      사용자가 러닝을 통해 만든 경로를 예술적인 이미지로 표현하는 러닝 아트 기능을 제공하며,
      이를 커뮤니티 공간에 업로드하여 다른 사용자들과 공유할 수 있도록 구현했습니다.

    • - 사용자 인증 및 데이터 저장

      JWT(JSON Web Token)를 기반으로 한 사용자 인증 시스템을 구현하여,
      간편 로그인을 통해 각 사용자의 데이터를 안전하게 분리 및 관리합니다.

전통주 소개 웹사이트

nodejs를 바탕으로 카카오 알림톡을 활용한 경북의 전통주를 소개하는 웹사이트를 만든 팀 프로젝트입니다.

  • 기간: 2025.06.16 ~ 2025.06.27
  • 역할: 프론트엔드 화면 구성
  • 사용 언어: HTML, CSS, JS, NODEJS, MONGODB
  • 기능: 전통주 소개, 성격 테스트, 카카오 알림톡 발송, 사용자 정보 저장 수집
    • - 전통주 정보 페이지 구성

      경상북도의 다양한 전통주 정보를 분류별(탁주, 약주, 증류주 등)로 나누어 소개하고,
      상세 페이지에서는 원재료, 도수, 양조장 정보 등을 함께 제공하여 사용자가 쉽게 이해할 수 있도록 구성했습니다.

    • - 전통주 추천 성격 테스트

      간단한 성격 테스트 문항을 통해 사용자의 취향을 파악하고,
      결과에 따라 어울리는 전통주를 카카오 알림톡으로 발송해줍니다.

    • - 카카오 알림톡 연동

      테스트를 완료한 사용자에게 결과 요약과 추천 전통주 정보를 카카오 알림톡으로 실시간 발송하는 기능을 구현했습니다. 카카오 비즈니스 인증과 메시지 템플릿을 활용해 안정적인 메시지 전송 시스템을 구성하였습니다.

파일 암호화 프로그램 및 통합 관리자 서버 개발

PC에서 동작하는 파일 암호화 프로그램과 그 동작을 통합 관리할 수 있는 관리자 서버를 구축했습니다. 추가적으로, 복호화 후 재암호화까지 가능한 Atool을 개발하여 보안성을 강화하였습니다.

  • 기간: 2025.04.29 ~ 2025.06.11
  • 역할: 백엔드 및 파일 암호화 로직 구현
  • 사용 언어: HTML, CSS, JS, PHP, MARIADB, C#
  • 기능: 파일 암호화/복호화, 관리자 권한 관리, 실시간 로그 기록, 세션 보안, IP 접근 제한 등
    • - 관리자 권한 및 IP 접근 제한

      최고 관리자와 중간 관리자로 역할을 구분하여 시스템 접근 권한을 설정하였고,
      관리자 접근 IP를 제한하여 외부 침입을 원천 차단하였습니다.

    • - 사용자 및 관리자 정보 조회/관리 기능

      관리자 페이지에서 사용자, 관리자, 부서, 권한 정보를 확인하고 수정/삭제할 수 있도록 구성했습니다.

    • - 실시간 로그 기록 및 세션 관리

      관리자 및 사용자 모든 작업에 대해 자동으로 로그를 생성하며,
      일정 시간(10분) 이상 작업이 없을 경우 세션을 종료하여 보안을 유지합니다.

    • - 파일 암호화 프로그램(i-Mon) 서버 연동

      사용자 등록, 로그인, 외부 반출 요청 등 다양한 작업 요청에 대해 서버가 API 형태로 응답하며,
      프로그램과 관리자 서버 간의 유기적인 통신을 지원합니다.

    • - 파일 복호화 및 재암호화 프로그램

      AES 알고리즘을 적용한 프로그램은 암호화된 파일을 복호화하여 작업 후 자동으로 재암호화하고,
      이 과정을 모두 로그로 기록하여 정보 유출을 방지합니다.
